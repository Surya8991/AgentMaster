## When to Use What

| Project Type | Recommended Stack |
|-------------|-------------------|
| SaaS App | Next.js + TypeScript + Tailwind + shadcn/ui + Zustand |
| Portfolio/Blog | Astro or Next.js + Tailwind + MDX |
| E-commerce | Next.js + Tailwind + Zustand + Stripe/Razorpay |
| Chrome Extension | Vanilla JS or React + Vite + Tailwind |
| Landing Page | Astro or Next.js + Tailwind + Framer Motion |
| Dashboard | Next.js + shadcn/ui + TanStack Query + Recharts |
