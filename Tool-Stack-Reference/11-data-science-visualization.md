## When to Use What

| Task | Recommended Tools |
|------|------------------|
| Quick Data Analysis | Pandas + Jupyter or Google Colab |
| React Dashboard | Recharts or Tremor + TanStack Query |
| Marketing Reports | Looker Studio + GA4 |
| Interactive Viz | Plotly or D3.js |
| Self-hosted BI | Metabase + PostgreSQL |
| Python Data App | Streamlit + Pandas + Plotly |
| Fast Data Processing | Polars or DuckDB |
