| Task | Recommended Tools |

| CWV Optimization | PageSpeed + WebPageTest + Web Vitals ext |
| Sitemap Setup | next-sitemap (Next.js) or Yoast (WordPress) |
| Meta Tags | Metatags.io + next-seo |
| Backlink Analysis | Ahrefs + Majestic |
| Index Monitoring | GSC + Screaming Frog |
