## When to Use What

| Project Type | Recommended Stack |
|-------------|-------------------|
| AI Chatbot | Claude/GPT API + Vercel AI SDK + Next.js |
| RAG App (Doc Q&A) | LlamaIndex + OpenAI/Claude + Supabase pgvector |
| Local AI Tool | Ollama + Llama 3 + Python |
| AI Writing Tool | Groq (fast) + Claude (quality) + Next.js |
| Multi-agent System | CrewAI or LangChain + Claude/GPT |
| AI Image App | Midjourney API or Stable Diffusion + Next.js |
| Code Assistant | Claude Code + AgentMaster |
