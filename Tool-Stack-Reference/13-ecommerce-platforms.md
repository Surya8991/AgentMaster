## When to Use What

| Project Type | Recommended Stack |
|-------------|-------------------|
| Quick D2C Store (India) | Shopify + Razorpay + Shiprocket |
| Custom Store (India) | Next.js + Razorpay + Medusa + Shiprocket |
| Global SaaS Store | Next.js + Stripe or Lemon Squeezy |
| Headless Commerce | Medusa/Saleor + Next.js + Algolia |
| WordPress Store | WooCommerce + Razorpay + Delhivery |
| Digital Products | Gumroad or Lemon Squeezy |
