## When to Use What

| Task | Recommended Tools |
|------|------------------|
| Landing Page (quick) | Carrd or Framer |
| Marketing Website | Webflow or Framer |
| MVP Web App | Bubble + Airtable + Zapier |
| Mobile App (no code) | FlutterFlow or Glide |
| Internal Admin Panel | Retool or Appsmith |
| Automation Workflow | n8n (free) or Zapier (easy) |
| Data Collection | Tally or Typeform |
