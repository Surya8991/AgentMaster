# Tech Reference: Game Development

| Name | Description | URL |
|------|-------------|-----|
| **Unity** | The dominant cross-platform game engine - 2D/3D, C#, mobile to console | unity.com |
| **Unreal Engine** | Epic's AAA game engine - C++/Blueprints, photorealistic rendering. | www.unrealengine.com |
| **Godot** | Free, fully open-source game engine - GDScript, C#, 2D/3D. | godotengine.org |
| **GameMaker** | YoYo Games' veteran 2D engine - GML scripting, indie favorite. | gamemaker.io |
| **Construct** | Browser-based 2D game maker - visual scripting, no code needed. | www.construct.net |
| **Defold** | Open-source 2D engine from King - Lua-based, mobile-first. | defold.com |
| **Phaser** | Popular HTML5 game framework - 2D web games in JS/TS. | phaser.io |
| **PixiJS** | Fast 2D WebGL renderer - used by game and visualization devs. | pixijs.com |
| **PlayCanvas** | Open-source WebGL game engine - collaborative cloud editor. | playcanvas.com |
| **Babylon.js Game** | Microsoft's WebGL/WebGPU 3D engine - full-featured for games and viz. | www.babylonjs.com |
| **Bevy** | Modern data-driven Rust game engine - ECS-first, fast iteration. | bevyengine.org |
| **MonoGame** | Open-source successor to XNA - cross-platform C# game framework. | www.monogame.net |
| **Cocos Creator** | Cocos Engine - Chinese-led cross-platform game engine, dominant in mob | www.cocos.com |
| **Ren’Py** | Visual novel engine - Python, vast indie following. | www.renpy.org |
| **RPG Maker** | Long-running RPG game engine - no-code dialogue and battle system. | www.rpgmakerweb.com |
